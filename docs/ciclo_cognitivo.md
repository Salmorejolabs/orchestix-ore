# Ciclo Cognitivo en ORE v0.1

El ciclo cognitivo es la unidad mínima de razonamiento del sistema. Cada ciclo sigue un flujo claro y trazable.

## Flujo del ciclo

1. Se crea un ContextSnapshot
2. El Scheduler ejecuta un ciclo
3. El Auditor inspecciona el estado
4. OCAP valida los artículos constitucionales
5. PrudenceEngine evalúa tensiones
6. PolicyEngine aplica políticas
7. MergeAgent fusiona decisiones
8. Memoria registra eventos
9. Se genera un Postmortem

Este flujo es fractal: cada rama puede generar subramas, todas gobernadas por la constitución.
