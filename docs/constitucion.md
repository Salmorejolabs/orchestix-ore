# Constitución de ORE v0.1 (Artículos I–VIII)

La constitución define los principios fundamentales que gobiernan cada ciclo cognitivo. Cada artículo es una regla explícita que puede ser consultada, validada y auditada.

## Artículos

1. Principio de Legalidad  
2. Transparencia del Proceso  
3. Prudencia Computacional  
4. No Daño  
5. Coherencia Interna  
6. Integridad del Ciclo  
7. Responsabilidad  
8. Gobernanza Fractal  

## Núcleo OCAP

El archivo ocap_core.py:
- carga los artículos
- los ordena
- permite consultarlos
- permite validarlos contra un ContextSnapshot

OCAP es el corazón constitucional del sistema.
